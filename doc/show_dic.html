Afficher le dictionnaire

Pour afficher l'ensemble des mots contenus dans le dictionnaire, il suffit d'aller dans edition->Voir le dictionnaire.

La fenêtre de dictionnaire:

Ajouter un mot au dictionnaire

Une fois dans le dictionnaire, il suffit de cliquer sur Ajouter pour ajouter un mot.

Une fois sur la fenêtre ci dessous, entrer le mot en français et en anglais, puis cliquer sur Ajouter.

Si un mot à plusieures traductions, il faut les entrer toutes séparées une virgule et un espace.

Il est également possible de le faire en allant dans edition->Ajouter des mots.

Supprimer un mot du dictionnaire

Une fois dans le dictionnaire, cliquer sur la ligne du dictionnaire correspondant au mot et cliquer sur Supprimer.

Rechercher la traduction d'un mot

Une fois dans le dictionnaire, il suffit de cliquer Rechercher sur pour rechercher la traduction d'un mot.

Une fois sur la fenêtre ci dessous, cocher le mode de recherche, puis entrer le mot et cliquer sur Go!.

Il est également possible de le faire en allant dans edition->Ajouter des mots.

Il est également possible de le faire en allant dans edition->Rechercher.
